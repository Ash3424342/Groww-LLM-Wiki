# Pay Pod

## Focus
First Pay transaction (Pay FPD) and Pay user retention. Cross-sell between Pay and Invest.

## Products
UPI payments, BBPS (bill payments)

## Key Economics

| Metric | Value |
|--------|-------|
| TTU resurrection: Invest content CAC | Rs.181 |
| TTU resurrection: Pay content CAC | Rs.287 |
| Non-TTU resurrection: Pay content CAC | **Rs.65** (vs Invest Rs.270) |
| Pay FPD CAC (Pay content) | Rs.5,284 |
| UPI CAC vs Bill rewards | UPI = **half** |
| Pay discovery (resurrected users) | Only 1/3 visit Pay |
| PreLoad vs Regular Cashback | PreLoad wins |

## Key Insight

- For **TTU resurrection**: use Invest content (cheaper)
- For **Non-TTU resurrection**: use Pay content (cheaper)
- **Default landing page fix needed**: Pay campaigns land on Invest page

## Retention Strategy

| Cohort | Strategy |
|--------|----------|
| BBPS Non-Bill (Fastag, MR) | Category-based; split by amount |
| BBPS Bill Category | Target bill **generation** date (not due date) |
| UPI users | **MCC Code**-based targeting by merchant type |
| Unused Cashback | Awareness → bill payment with unused balance |
| Pay Only users | Cross-sell nudge to start investing |
