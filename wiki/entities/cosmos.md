# COSMOS
Journey/automation system for WhatsApp flows. WA onboarding journeys are built and run here.
## What's Known
- WA KYC drop-off journeys implemented in COSMOS (live for 100% of users)
- ~200 incremental Esigns/day via COSMOS journeys
