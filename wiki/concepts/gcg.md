# GCG — Global Control Group

Permanent holdout from all promotional comms (only transactional messages). 30-day holdout window. Baseline for measuring incremental uplift.

## Sizing by Lifecycle

| Stage | GCG Size |
|-------|----------|
| New users (Activation) | 2-5% |
| Active users (RID) | 5-10% |
| Churn-risk users | 5-8% |
| Blended average | ~5% |

The "5%" in campaign docs is the blended average, not a fixed rate.
